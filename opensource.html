オープンソースについて

VistaUpdater コミュニティのみなさんが、VistaUpdaterを改善するために、快適に使用するためにVistaUpdaterをオープンソースにしています。
サイドバーのGitHubからオープンソースの製品を閲覧することができます。

オープンソースにしているソフトウェア一覧

VistaUpdater: VistaUpdaterのメインプログラムのソースコード。
Win7Updater: Win7Updaterのメインプログラムのソースコード。
修復ツール: VistaUpdater 修復ツールのソースコード。
Windows Update UI Patcher: WU UI Patcherのソースコード。
