ツールをダウンロード

このページは何ですか？

Windows VistaとWindows 7 用のツールをダウンロードすることができます。(.NET Framework 4.6, ESU patcher, etc..)

Windows Vista

Windows Vistaで使用できるツールリスト

VistaUpdaterによって作成されたツール

Server 2008 Update Patcher

Windows Server 2008 SP2 の更新プログラムとESUの更新を受け取るためのツールキットです。
サポートOS: Windows Vista SP2 / Windows Server 2008 SP2 (ESUパッチのみ)

バージョン 1.1 ダウンロード


VistaUpdater 修復ツール

VistaUpdater で破壊されたPC(VUブリックされたPC)を修復します。
サポートOS: Windows Vista SP2

バージョン 1.2.1 ダウンロード


Windows Update UI Patcher

Windows Vista (VistaUpdater適用済み)とWindows Server 2008のWindows UpdateのUIをWindows Vistaの最新WUAのデザインに変更します。
サポートOS: Windows Vista SP2 / Windows Server 2008 SP2 (ESUパッチのみ)

バージョン 1.0 ダウンロード


Windows Vistaのランタイム/SPなど

.NET Framework 4.6

.NET Framework 4.6 のオフラインインストーラーです。
サポートOS: Windows Vista SP2 ~ 8.1, Server 2008 SP2 - 2012 R2

ダウンロード


Windows Vista Service Pack 2

Service Pack 2を手動でインストールするには、これを使用してください。
サポートOS: Windows Vista SP1 / Windows Server 2008 SP1

ダウンロード


Windows Vista Service Pack 1

Service Pack 1を手動でインストールするには、これを使用してください。
サポートOS: Windows Vista RTM
