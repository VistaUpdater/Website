ツールをダウンロード

このページは何ですか？

Windows VistaとWindows 7 用のツールをダウンロードすることができます。(.NET Framework 4.6, ESU patcher, etc..)

Windows Vista

Windows Vistaで使用できるツールリスト

VistaUpdaterによって作成されたツール

Server 2008 Update Patcher

Windows Server 2008 SP2 の更新プログラムとESUの更新を受け取るためのツールキットです。
サポートOS: Windows Vista SP2 / Windows Server 2008 SP2 (ESUパッチのみ)

バージョン 1.1 ダウンロード


VistaUpdater 修復ツール

VistaUpdater で破壊されたPC(VUブリックされたPC)を修復します。
サポートOS: Windows Vista SP2

バージョン 1.2.1 ダウンロード


Windows Update UI Patcher

Windows Vista (VistaUpdater適用済み)のWindows UpdateのUIをWindows Vistaの最新WUAのデザインに変更します。
サポートOS: Windows Vista SP2 X64 Edition

バージョン 1.0 ダウンロード
